# Genesis Roadmap

## v1.0 — Launch (June 2026)

### Core Skills
- **anchor** — Checkpoint system for tracking progress
  - Save/load checkpoint state
  - Track what was done and by whom
  - Resume from checkpoint instantly
  
- **anchor-handoff** — Cross-model context transfer
  - Package session state for other models
  - Seamless hand-off to Opus/Sonnet
  - Preserve memory and context

- **anchor-memory** — Persistent learning
  - Remember important patterns
  - Recall across sessions
  - Build knowledge incrementally

- **anchor-security** — Safety guardrails
  - Validate operations before executing
  - Prevent common mistakes
  - Audit trail for compliance

### Launch Checklist
- [ ] All 4 skills production-ready
- [ ] Documentation complete
- [ ] Installation tested on Claude Code 2.1+
- [ ] GitHub repo public
- [ ] First community feedback incorporated
