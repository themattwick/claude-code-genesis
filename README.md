# Genesis — Production-Ready Skills for Claude Code

**The origin of intelligent workflows.**

Genesis is a comprehensive hub of production-ready skills for Claude Code. Start with intelligent checkpoints, cross-model handoffs, and persistent memory—then expand to security guardrails, research automation, and beyond.

## What Is Genesis?

Genesis provides battle-tested skills that teach Claude Code to handle real development workflows. Each skill is:
- ✅ **Production-ready** — evolved from intensive daily use
- ✅ **Focused** — solves one problem well
- ✅ **Documented** — clear patterns and examples
- ✅ **Testable** — built for real projects

## Quick Start

### Installation

\`\`\`bash
# Install the entire genesis hub
/plugin install genesis

# Or install specific skills
/skill install genesis@anchor
/skill install genesis@anchor-handoff
/skill install genesis@anchor-memory
\`\`\`

### First Steps

1. **Anchor** — Checkpoint system for long-running tasks
2. **Anchor Handoff** — Transfer context between models
3. **Anchor Memory** — Persistent learning across sessions
